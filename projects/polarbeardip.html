Polar Bear Mobile Event Progressive Web App

Nov 2019

Polar Bear Dip is a mobile event progressive web application which the user doesn't need to download it from the app store, and it supports offline service. The app supports interactive maps, mini polls/surveys, photo galleries schedule checks, video player and event location, etc.

Design Thinking

We organized lots of custom event every year in World Vision Canada. To better engage the corresponding activities, we need an application to do publicity and introduction. Along with my teammates, I researched the feasibility of implementing this application either through a progressive web app or a native mobile app. After extensive research we determined that a progressive web application would be the best way to engage events. The reason for this is that most of the people do not like to download an application for just one event per year. PWA can add to the home screen by only one single click and looks like a real native app.
