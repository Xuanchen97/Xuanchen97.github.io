Polar Bear Mobile Event Progressive Web App

Nov 2019

Polar Bear Dip is a mobile event progressive web application that users don't need to download from an app store, and it supports offline service. The app features interactive maps, mini polls/surveys, photo galleries, schedule checks, a video player, and event location information, among other features.

Design Thinking

At World Vision Canada, we organize numerous custom events every year and to better engage with these activities, we need an application for publicity and introductions. With my team, I researched the feasibility of implementing this application either through a progressive web app or a native mobile app. After extensive research, we determined that a progressive web application would be the best way to engage with events. This is because most people do not want to download an application for just one event per year. A PWA can be added to the home screen with just one click and looks like a real native app.
