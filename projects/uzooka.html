Uzooka (Shopify Platform)

Jan 2019

Uzooka is an American Company mainly selling Components for Motor vehicle. They have over 5,000 products selling online. I participated in part of this project, and the following are the main tasks I did.

Microsoft Bing spellcheck API

Implemented the Microsoft Bing spellcheck API to spellcheck values as they are being typed into the header search field (see attached screenshot). If a correction is returned by the Spell Check API, add the "suggested" spelling at the top.

Filter Catalog

The filter was too long due to there are hundreds of car models. The decided solution is that when the user selects the brand of the car, only the model of the corresponding brand displayed in the filter.

Headlights Placement graphic meta implementation

The business requirement is to display the headlight/tail light placement graph to the product page depends on the car model and stock quantity. We used Shopify's own programming language Liquid to get the meta data for the current product being shown, determine what the values are, then display the correct image.

Development Tools/Languages

  • Shopify Liquid
  • MS Bing Spellcheck API
  • JavaScript
  • jQuery
  • PHP & MySQL
